**Title**:
Phase 3: Background scheduler span instrumentation

**Description**:
Add root trace spans to the Poller scheduler so that scheduled background ingestion activity is independently observable in SigNoz, decoupled from HTTP request traffic. Each poller tick produces a root `scheduler.poll.tick` span; each source processed within a tick produces a `scheduler.poll.source` child span that parents the pipeline spans from Phase 2.

## Requirements
- Add module-level `tracer = trace.get_tracer(__name__)` in `scheduler/poller.py`
- Wrap `_tick()` body in a root span `scheduler.poll.tick` (context=None or use_span with no_attach=False so it becomes a root) with attributes: `sources_due_count`, `adapters_registered`
- Wrap the per-source body inside `_tick()` in a `scheduler.poll.source` child span with attributes: `source_id`, `adapter_id`
- For `trigger_immediate_ingest()` (called from the adapter reset route), create a root span `scheduler.poll.immediate` — do not inherit the HTTP request context since the HTTP request returns 202 immediately
- On per-source exception: `span.set_status(StatusCode.ERROR)`, `span.record_exception(exc)`, `span.set_attribute("error.consecutive_failures", count)`
- FR-8, US-7

## Design Guidance
Implements Phase 3 (Scheduler and Background Task Spans) from the architecture design. The span structure for `_tick()` and `trigger_immediate_ingest()` is defined in the architecture document. The root span context propagation note (background thread, no HTTP parent) is described in the Phase 3 section.

## Acceptance Criteria
- [ ] After the poller executes a scheduled tick, a root `scheduler.poll.tick` span appears in SigNoz with no HTTP parent
- [ ] The tick span contains a `scheduler.poll.source` child span for each source processed, which in turn parents the `pipeline.ingest` spans from Phase 2
- [ ] `adapter_id` and `trigger` attributes are present on scheduler spans
- [ ] When a scheduler-triggered ingestion fails, the span status is ERROR
- [ ] `trigger_immediate_ingest()` produces a root span not parented to the initiating HTTP request
