**Title**:
Phase 5: Replace dashboard StatCard with Heimdall StatTile and StatGrid

**Description**:
Replace the custom `StatCard` component in the dashboard index route (`index.tsx`) with Heimdall's `StatTile` and `StatGrid` components. Domain accent colors must be applied to each tile. The `DomainBreakdown` and `ActivityFeed` sections should be wrapped with Heimdall `Panel` for consistent surface styling.

## Requirements
- Remove the custom `StatCard` component definition from `index.tsx`
- Replace all `StatCard` usages with Heimdall `StatTile` components composed inside a `StatGrid` wrapper
- Map `StatCard` props to `StatTile` equivalents: `label`, `value` are direct mappings; `icon` has no `StatTile` prop — if icon display is needed, use `className` or `children` on the tile
- Apply each domain's accent color to its tile via inline `style` prop (since `StatTile.color` accepts only a 4-value union that does not cover all 9 domain colors)
- Wrap `DomainBreakdown` and `ActivityFeed` sections with Heimdall `Panel` components for consistent dark-canvas surface treatment
- Domain color values must come from the CSS custom properties registered in Phase 4

## Design Guidance
Implements the Dashboard Enhancement phase and Pattern 3 (Dashboard StatCard → StatTile) from the architecture design. The architecture explicitly notes that `StatTile.color` only accepts a fixed 4-color union, requiring inline `style` for domain colors, and that `StatGrid` replaces the current `grid grid-cols-2 lg:grid-cols-4 gap-4` pattern.

## Acceptance Criteria
- [ ] The dashboard index route renders with no custom `StatCard` component; all stat displays use Heimdall `StatTile` within a `StatGrid`
- [ ] Each domain stat tile displays that domain's accent color correctly
- [ ] Domain color values resolve from `--domain-*` CSS custom properties, not hardcoded hex literals
- [ ] `DomainBreakdown` and `ActivityFeed` sections render within Heimdall `Panel` wrappers
- [ ] Dashboard layout is visually consistent with the dark-canvas design on all breakpoints
